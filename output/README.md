# Output

The output folder contains the results of the training and evaluation of the model. It is organized as follows:
- `runs/`: Contains the saved models and training logs for each run. Each run is stored in a separate folder
- `plots/`: Contains the plots generated during training and evaluation, such as loss curves and sample inpaintings.
- `rapsd/`: Contains the RAPSD scores calculated during evaluation.
- `real_samples/`: Contains the real samples used for reference during evaluation.
- `evaluation/`: Contains the plots produced during the evaluation, such as the plots of metrics over time.
